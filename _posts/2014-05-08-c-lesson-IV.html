  1. 函数指针 
    声明一个函数：
int strcmp(char *s1, char *s2);

int (*comp)(void *, void *)
其中 comp 代表一个函数指针的占位符，是一个指向有两个 void* 参数，且返回值是int类型的函数的指针，
而 strcmp 该名称则为真正的函数指针；

对函数指针的使用：
if ((*com)(arg1, arg2) < 0)

函数指针的类型转换：
(int (*)(void *, void *))strcmp
  2. char **argv; 
    argv是一个指针，该指针指向一个指针A，而指针A则是一个类型是char*的指针（即该指针指向一个char，通常用于指向一个字符串），argv通常用于声明一个字符串数组；
  3. int (*daytab)[13]; 
    *daytab是一个有13个元素的数组，数组里的每一个元素是一个int，而daytab则是指向这个数组的指针；
